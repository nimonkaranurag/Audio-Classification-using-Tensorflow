2.<placeholder>
	a. FSDKaggle2018.audio_test contains all the test audio(.wav) files
	b. FSDKaggle2018.meta contains two .csv files
		a. "train_post_competition.csv" contains the training labels
		b. "test_post_competition_scoring_clips.csv" contains the test labels
	c. FSDKaggle2018.audio_train contains all the training audio(.wav) files

3.Download the dataset and upload it to your google drive, replace the __dir__ variables in the main and evaluation scripts to reproduce results. Make sure to replicate experiment on a high RAM system or adjust the sample_rate accordingly.
